## Description

Repository for storing notes, papers, and other materials related to Social Network Analysis (SNA) by me.

This repository serves as a centralizer of my SNA-related knowledge and discoveries, you're free to use anything here as you wish. Be aware that this is a personal repository and may contain informal notes and ideas as well as not-accurate information on the subject. Always refer to official sources for accurate information.

You can also reproduce any code or analysis found here, I will always credit the original authors for code that I didn't write myself.
